HOF==>Heigher Order Function ==>Heigher Order Function that operate on other function either by taking them as argument or by returning them. ==>forEach, map, filter, reduce ==>forEach==>It execute provided function once for each array element. ==>map==>it create array. The final array will have same number of element as that of calling array. ==>filter==>it create array. It will only the element which satisfy the condition. ==>reduce==>it create single output.It execute a reducer function for each array elemement.
